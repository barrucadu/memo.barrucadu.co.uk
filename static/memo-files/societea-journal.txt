; This is a simple text format for double-entry accounting, from which
; nice  reports  can  be  generated with  the  programs  "ledger"  and
; "hledger". I started using hledger to track my finances this summer,
; and  quite liked  it; so  I've  decided to  do the  same for  teasoc
; (rather than that spreadsheet).
;
; The format should be fairly self-explanatory. A transaction consists
; of a  date, a title, and  some postings. A posting  names an account
; (account names can  have spaces in, such as  "assets:own funds") and
; gives the amount. If the amount  is missing, it is inferred from the
; other postings to make the entire transaction balance.
;
; A brief explanation of the workflow: someone buys something for the
; society, and that money moves from "liabilities" to "expenses"; when
; a claims form is handed in, that liability is paid off by moving
; money from "assets".
